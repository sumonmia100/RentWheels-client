## ðŸ“‹ Project Overview
**RentEase** is a modern car rental platform where users can browse, book, and manage cars easily.  
Providers can list their cars for rent, and customers can book available cars securely with authentication.

---

## ðŸŒŸ Main Features

- **Firebase Authentication**
  - Secure Google login and logout system for users and providers.

- **Dynamic Car Listings**
  - Cars are fetched from **MongoDB** and displayed with real-time availability.

- **My Listings (Private Route)**
  - Logged-in providers can view, update, or delete their own cars.

- **Booking System**
  - Users can book a car, and the car status automatically updates to *Unavailable*.

- **Search & Filter**
  - Instantly search cars by name, brand, or category from the home page.

- **JWT Authorization**
  - Protected API routes for private data (like My Listings, Bookings, etc.).

- **Customer Testimonials & Top Rated Cars**
  - Beautiful extra sections that enhance user trust and UI appeal.
